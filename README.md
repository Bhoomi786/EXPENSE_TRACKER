# Expense Tracker Pro

A modern GUI-based Expense Tracker application built using Java Swing.

## Features

- Add Expenses
- Delete Expenses
- Total Expense Calculation
- Category Selection
- Permanent File Storage
- Modern GUI Design
- Light and Dark Theme Support

## Technologies Used

- Java
- Swing
- File Handling
- Git & GitHub

## How to Run

1. Compile the program

```bash
javac Main.java
```

2. Run the program

```bash
java Main
```
